###Ruby Bits学习笔记    
    
    1. 在Ruby中, nil 和 false 是等价的
    2. "" 和true是等价的
       0  和true是等价的
       [] 和true是等价的
___
    attr_accessor 和 attr_reader均用在类class的定义中

#####attr_accessor        
    attr_accessor :name
    表示name可读可写
    等价于如下两个方法        
    def name=(value)           
       @name=value    
    end
    def name
       @name
    end
#####attr_reader
    attr_reader :age
    表示 age是只读属性
    等价于
    def age
      @age
    end
